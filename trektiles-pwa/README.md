# TrekTiles PWA

Générateur de cartes hors-ligne PMTiles pour randonnée et trekking.

## Installation
1. Ouvrir `index.html` dans un serveur local (HTTPS requis)
2. Cliquer sur le bouton "Installer" dans la barre d'adresse
3. L'app s'installe comme une application native

## Fonctionnalités
- Génération PMTiles depuis OSM et IGN
- Sélection de zone par carte ou coordonnées
- Choix du dossier de sauvegarde
- Fonctionne hors ligne après installation
