Import des données existantes
Pour modifier les données de cenoutil, importez les fichiers JSON correspondants depuis le dossier cenoutil/data/. Les modifications se font ensuite dans les onglets dédiés, puis exportez le bundle final.
Workflow de mise à jour
1. Importez les JSON actuels de cenoutil (ci-dessus) ou laissez cenoprepa les charger automatiquement si les deux PWA sont dans le même dossier PWA/.
2. Modifiez les données dans les onglets Observations, Accès falaise, Fossiles, Pages info.
3. Exportez le bundle ZIP → décompressez dans cenoutil/ sur GitHub.
4. Les photos doivent être placées manuellement dans cenoutil/photos/.

Export du bundle cenoutil
Export complet — bundle ZIP
Génère une archive ZIP contenant tous les fichiers JSON mis à jour (markers.json, observations.json, fossiles.json, pages.json). Décompressez l'archive et copiez le dossier data/ dans cenoutil/ sur GitHub Pages.
Exports individuels
Télécharger un seul fichier JSON pour une mise à jour partielle.
Note sur les photos
Les photos de terrain ne sont pas incluses dans le bundle JSON. Elles doivent être copiées manuellement dans le dossier cenoutil/photos/ du dépôt GitHub, dans les sous-dossiers correspondants :

photos/observations/ — photos des marqueurs d'observation
photos/fossiles/ — photos des espèces du catalogue
photos/pages/ — images des pages d'information
photos/ — photos des 13 accès falaise
