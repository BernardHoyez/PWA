
# Randonne PWA

Application web progressive pour suivre une randonnée avec fond de carte IGN Plan V2.

## Déploiement

Déployer sur GitHub Pages dans le dépôt `BernardHoyez.github.io` sous le chemin `/PWA/randonne`.

## Fonctionnalités

- Fond de carte IGN Plan V2 sans clé API
- Suivi GPS en temps réel
- Sauvegarde locale du tracé
- PWA installable
