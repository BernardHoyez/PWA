# randonne - PWA de randonnée

## Installation locale
- Cloner ce dépôt
- Ouvrir index.html dans un serveur local (ex: `npx serve`)

## Déploiement GitHub Pages

1. Pousser tous les fichiers dans votre dépôt GitHub.
2. Dans les paramètres du dépôt, activer GitHub Pages sur la branche principale.
3. Configurer la racine comme dossier source.
4. Accéder à l'URL: https://BernardHoyez.github.io/PWA/randonne/

---

Les icônes `icon192.png` et `icon512.png` doivent être placées à la racine du dossier `randonne`.
