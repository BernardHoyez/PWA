**mestiles** est un générateur de fichier .mbtiles
Les tuiles sont importées d'OSM (open Street Maps)
On peut choisir le rayon en km autour d'un point central
Le zoom est limité entre 12 et 16
On peut choisir de sauvegarder dans un sous-dossier de Downloads et Documents
