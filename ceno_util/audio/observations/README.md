# Dossier audio/observations/

Placer ici les fichiers audio MP3 (ou WebM/M4A) correspondant aux
commentaires vocaux des fiches d'observation.

Les noms de fichiers doivent correspondre exactement aux valeurs
du champ "audio" dans data/observations.json.

Exemple :
  observations.json : "audio": "audio/observations/obs001.mp3"
  → Fichier attendu   : audio/observations/obs001.mp3

Format recommandé : MP3, mono, 64 kbps, max 2 min (~1 Mo)
