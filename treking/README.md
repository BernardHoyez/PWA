# Treking PWA

Application de tracé GPS avec fonds de carte PMTiles hors-ligne.

## Déploiement sur GitHub Pages

1. Créez un dépôt `BernardHoyez.github.io`
2. Placez les fichiers dans un dossier `PWA/treking/`
3. Activez GitHub Pages dans les settings
4. URL : https://BernardHoyez.github.io/PWA/treking

## Utilisation

1. **Mode En Ligne** : OSM ou IGN direct
2. **Mode Hors Ligne** : Sélectionnez votre fichier `.pmtiles` (généré avec TrekTiles)
3. Appuyez sur **START** pour commencer l'enregistrement
4. **STOP** pour terminer
5. **ENREGISTRER** pour exporter GPX ou KML

## Fonctionnalités
- Fonctionne 100% hors-ligne après installation
- Enregistrement GPS précis
- Export GPX/KML compatible avec Garmin, QGIS, etc.
- Interface mobile-first
- Économie de batterie (GPS uniquement en enregistrement)
