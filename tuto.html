Installer une PWA
sur votre appareil

Une Progressive Web App s'installe en quelques secondes — sans passer par un store — sur tous vos appareils.

🤖 Android 🍎 iOS / iPadOS 🪟 Windows 💻 macOS

Pourquoi installer une PWA ?

🚀

Lancement instantané

Accès direct depuis l'écran d'accueil, sans passer par le navigateur.

📡

Fonctionne hors-ligne

Le contenu est mis en cache et reste accessible sans connexion.

🔔

Notifications push

Recevez des alertes en temps réel, comme une app native.

🔒

Sécurisée

Toujours servie en HTTPS, sans risque de malware via un store tiers.

🤖

Android

Chrome pour Android · Android 5.0+ recommandé

1

Ouvrez le site dans Chrome

Naviguez vers l'URL de la PWA. Chrome affiche une icône ⊕ bleue dans la barre d'adresse dès que l'app est installable.

2

Appuyez sur « Ajouter »

Une fiche apparaît en bas de l'écran avec le nom et l'icône de l'app. Confirmez en appuyant sur Ajouter.

3

L'app est sur votre écran

Un raccourci apparaît sur l'écran d'accueil. L'app s'ouvre désormais en plein écran, sans interface navigateur.

💡 Si l'icône ⊕ n'apparaît pas, ouvrez le menu ⋮ (trois points) puis choisissez « Ajouter à l'écran d'accueil ».
🍎

iOS / iPadOS

Safari uniquement · iOS 16.4+ pour les notifications push

1

Ouvrez Safari et naviguez

Accédez au site dans Safari (obligatoire sur iOS). Appuyez ensuite sur l'icône ⬆ Partager dans la barre d'outils du bas.

2

Choisissez « Sur l'écran d'accueil »

Faites défiler les options et sélectionnez Sur l'écran d'accueil (icône ⊞). Elle apparaît dans la rangée du bas de la feuille de partage.

3

Nommez et confirmez

Un champ éditable vous permet de renommer le raccourci. Appuyez sur Ajouter en haut à droite pour terminer.

⚠️ Sur iOS, seul Safari permet d'ajouter une PWA à l'écran d'accueil. Chrome et Firefox iOS ne le supportent pas encore.
🪟

Windows

Microsoft Edge (recommandé) · Chrome · Windows 10 / 11

1

Repérez l'icône d'installation

Dans Microsoft Edge, une icône ⊕ apparaît à droite de la barre d'adresse quand le site est installable. Cliquez dessus.

2

Confirmez l'installation

Une fenêtre s'ouvre avec les options : épingler dans la barre des tâches, créer un raccourci sur le Bureau… Cochez vos préférences et cliquez Installer.

3

L'app est installée

La PWA s'ouvre dans sa propre fenêtre et son icône apparaît dans la barre des tâches et/ou sur le bureau selon vos choix.

💡 Avec Chrome, cliquez sur ⋮ → Enregistrer et partager → Installer la page en tant qu'application si l'icône automatique n'apparaît pas.
💻

macOS

Safari 17+ · macOS Sonoma (14) ou ultérieur recommandé

1

Ouvrez le site dans Safari

Accédez à la PWA dans Safari. Une icône ⊕ peut apparaître dans le champ d'adresse si le site le supporte.

2

Menu Favoris → « Sur l'écran d'accueil »

Dans la barre de menus, allez dans Favoris → Ajouter à l'écran d'accueil… (ou cliquez sur l'icône ⊕ si présente).

3

L'app apparaît dans le Dock

La PWA s'installe et son icône est ajoutée au Dock. Elle s'ouvre dans sa propre fenêtre sans interface Safari.

🆕 Cette fonctionnalité est disponible depuis macOS Sonoma (14) avec Safari 17. Sur les versions antérieures, utilisez Chrome ou Edge.
