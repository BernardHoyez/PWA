06 / Installation & Protocole

Setup côté Android (Xiaomi 14T Pro)

  1. Installer FocusBrk Server (APK fourni) ou utiliser Tasker + AutoRemote
  2. Activer les permissions caméra
  3. S'assurer que le téléphone et le PC sont sur le même réseau Wi-Fi
  4. Lancer le serveur → noter l'IP affichée
  5. Entrer cette IP dans le champ connexion ci-dessus

Protocole WebSocket (JSON)

  • CAPTURE {"cmd":"capture","focus":0.45,"iso":"auto"}
  • PING {"cmd":"ping"}
  • STATUS {"status":"ok","shot":3}

La valeur focus va de 0.0 (minimum) à 1.0 (infini). Le serveur Android utilise l'API Camera2.
